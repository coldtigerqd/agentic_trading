# Trade Risk Command

分析当前持仓的风险状况。

请运行以下脚本并解释结果：

```bash
python runtime/trade_risk.py
```

如果用户需要自定义风险阈值，支持：
- `--threshold=0-100` - 风险警告阈值（默认70）

此命令会分析：
- ✅ 持仓总数和总风险敞口
- ✅ 风险评分（0-100）
- ✅ 需要关注的高风险持仓
- ✅ 具体的行动建议

**风险等级**:
- 🟢 低风险（0-40）：可正常交易
- 🟡 中等风险（41-70）：需要监控
- 🔴 高风险（71-100）：建议减仓或对冲

运行完成后，请向用户总结：
- 当前风险评分和等级
- 高风险持仓详情
- 建议的风险管理措施

**注**: 此命令需要通过 MCP 访问 IBKR 持仓数据。
