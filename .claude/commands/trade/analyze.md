# Trade Analyze Command

执行完整的7步交易分析工作流。

请运行以下脚本并解释结果：

```bash
python runtime/trade_analyze.py
```

如果用户需要自定义参数，支持以下选项：
- `--sector=tech|finance|healthcare|energy|ALL` - 指定板块
- `--confidence=0.0-1.0` - 置信度阈值（默认0.75）
- `--max-signals=1-20` - 最大信号数（默认10）
- `--prepare-only` - 只分析不执行
- `--force-sync` - 强制同步数据

**工作流步骤**:
1. 市场健康检查（数据质量、市场状态）
2. 账户状态获取（净值、可用资金）
3. 持仓风险分析（风险评分、集中度）
4. 蜂群智能分析（并发咨询多个实例）
5. 信号过滤（按置信度和板块）
6. 风险评估（Kelly仓位计算、投资组合约束）
7. 执行决策（准备模式或实际下单）

运行完成后，请向用户总结：
- 市场状态和数据质量
- 找到的高置信信号
- 风险评估结果
- 执行建议
